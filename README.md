📌 Hostel Outpass System
📖 Overview

The Hostel Outpass System is a web-based application that digitizes the process of requesting and managing hostel outpasses.

It replaces manual registers with a structured system where students can apply for permission and authorities can review, approve, or reject requests efficiently.

🌐 Live Demo

🔗 Deployment Link:
👉 https://hostel-outpass-red.vercel.app


🚀 Features

👨‍🎓 Student
Submit outpass requests with reason and duration
Track request status (Pending / Approved / Rejected)
View previous outpass history

🛡️ Admin / Warden
View all student requests in one dashboard
Approve or reject applications
Maintain records of outpass activity

⚙️ System
Secure authentication using Supabase
Real-time database updates
Clean and responsive interface

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend / Database: Supabase (PostgreSQL + Auth)
Deployment: Vercel
Version Control: Git & GitHub
